น้องๆอ่านหน่อยนะ

สวัสดี พี่ชื่อ ดี นะ เป็นนักเรียนสายการเรียนคอมพิวเตอร์ Gen 4นะ

อันนี้เป็นเว็ปไซด์แจก Source code ที่พี่ศึกษามาและเคยลองฝึกทำพี่หวังว่าน้องๆที่เข้ามาจะได้นำสิ่งที่พี่ศึกษาหามาไปพัฒนาตนเองนะและน้องๆที่ยังแข่งขันศิลปหัตกรรม พี่ขอโทษนะที่ไม่มีเวลาว่างไปช่วยสอนหริอให้คำแนะนำนะพี่หวังว่าสิ่งนี้จะเป็นประโยชน์นะ

ขอกำลังใจให้พี่หน่อย

พี่จะยื่น Portfolio สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกะบัง ที่นี่เป็นมหาลัยอันดับ 1 ในใจพี่และพี่หวังว่าพี่จะติดที่แห่งนี้แต่ถ้าไม่ติดปีหน้าพี่ก็จะมาอีกไม่แน่เราอาจได้เป็นคู่แข่งกัน

ถ้าใครยังคิดไม่ออกจะเรียนมหาลัยไรดี พี่ก็อยากจะแนะนำ สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกะบัง เอาไว้นะดีทุกคณะลองมาดูกันได้แถมสังคมและบรรยากาศยังดีด้วย
